# Kabu Sim

Next.js で構築されたシンプルな SaaS サービスです。

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## プロジェクト構成

- `app/` - Next.js App Router のディレクトリ
  - `layout.tsx` - ルートレイアウト
  - `page.tsx` - ホームページ
  - `globals.css` - グローバルスタイル

## 使用技術

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## スクリプト

- `npm run dev` - 開発サーバーを起動
- `npm run build` - プロダクションビルド
- `npm run start` - プロダクションサーバーを起動
- `npm run lint` - ESLint を実行
